# 🎴 Yu-Gi-Oh! Card Browser (Angular)

Aplicación en Angular 20 para buscar y visualizar cartas de Yu-Gi-Oh! usando la API pública de [YGOPRODeck](https://db.ygoprodeck.com/api-guide/).

## 🚀 Características
- Búsqueda de cartas por nombre.
- Infinite scroll para cargar más resultados.
- Vista de detalle con imagen, descripción y precios de mercado.
- Diseño responsive con Angular standalone components.

## 🛠️ Tecnologías
- Angular 20 (standalone)
- RxJS (async pipe, valueChanges)
- ngx-infinite-scroll
- API YGOPRODeck
